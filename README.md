# Лаба по Python №4 "Гуси, ОПГ и казино"

## Установка и запуск
```bash
uv venv

source .venv/bin/activate # Для Linux/MacOS
source .venv\Scripts\activate # Для Windows

uv sync
```
```bash
uv run -m src.main --steps <int> --seed <int>
```
* steps - количество шагов симуляции
* seed - сид (логично)

## Тесты
```bash
uv run pytest
```

# Что да как
`src/goose.py`
Тут обитают базовый гусь с именем и полем под ОПГ. От него наследуются боевой и кричащий гусь.
Тут же присутствует класс ОПГ гусей. Механизм закидывания в ОПГ:
1. Если первый гусь в ОПГ, то второй вступает туда же.
2. Если оба без ОПГ, то создаётся новое.

Гуси могут нападать и кричать как в одиночку, так и целым ОПГ (шанс нападения целого ОПГ 20%).
1 уровень крика гуся увеличивает панику(стресс, ...) на 1 уровень.
1 уровень мощи(силы, ...) весит 5% ворованных денег, а 1 уровень стресса 2%.

В `src/player.py` класса игрока-лудомана. Также тут высчитывается шанс неудачной крутки: 60% + 2% * уровень стресса

В `stc/chip.py` класс фишек.

`*_collections.py` и `src/casino_balance.py` коллекции

`casino.py`. Инициализация всех коллекций, управление счетами, запуск всех функций

`simulation.py` Запуск симуляции

Бонус: если гусь отбирает последние деньги, лудоману не на что кушать, он умирает
Бонус: ОПГ могут устроить стычку между собой
